## Списки

Чтобы добавить ненумерованные списки, то необходимо выделить пункты звёздочкой (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, то необходимо просто пронумеровать. Например, вот так:
1. Элемент 1
2. Элемент 2
3. Элемент 3
